## Supported Simulators

NovyWave works with waveform files from any simulator that outputs standard formats:

| Format | Extension | Simulators |
|--------|-----------|------------|
| **VCD** | `.vcd` | Icarus Verilog, Verilator, ModelSim, and most Verilog/SystemVerilog simulators |
| **FST** | `.fst` | Verilator, GTKWave-optimized output |
| **GHW** | `.ghw` | GHDL (VHDL simulator) |
