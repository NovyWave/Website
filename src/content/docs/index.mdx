## Features

<CardGrid>
	<Card title="Standard Formats" icon="document">
		Works with **VCD**, **FST**, and **GHW** waveform files, so the docs can stay focused on the files you open instead of long simulator lists.
	</Card>
	<Card title="Desktop First" icon="laptop">
		Native apps for **Linux**, **macOS**, and **Windows** are the main path. There is also a browser mode when running NovyWave locally or on another machine.
	</Card>
	<Card title="Shortcut Support" icon="rocket">
		Keyboard shortcuts cover common work such as zooming, panning, cursor movement, and jumping between transitions without pretending every action is keyboard-only.
	</Card>
	<Card title="Multi-File Comparison" icon="list-format">
		Load multiple waveform files in one session and compare signals across regression runs, design variants, or separate parts of the system.
	</Card>
</CardGrid>

## Who is NovyWave For?

- **Hardware engineers** working on digital design verification
- **Students** learning digital design and HDL simulation
- **Hobbyists** exploring FPGA and ASIC development
- **Researchers** analyzing simulation results

## What This Site Covers

- End-user installation and workflow guides
- Practical tutorials using real HDL examples
- Development setup and architecture notes
- Internal protocol and data-type reference for contributors
